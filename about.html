About blocq

blocq is a modern multiplayer block game that allows players to compete in real-time.

Features

  • Single player mode
  • Real-time multiplayer
  • Competitive gameplay
  • Modern visual design

How to Play

Desktop: Use arrow keys to move and rotate pieces. Press space to hard drop.

Mobile: Use on-screen controls - hold left/right buttons to move, tap rotate button to rotate, and double-tap drop button for hard drop.

Development

blocq was developed using JavaScript, HTML5 Canvas, and WebSockets for real-time multiplayer functionality.
